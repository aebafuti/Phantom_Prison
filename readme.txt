『幽獄の14日間』ver1.13b
　最終更新：2016/5/22 制作：饗庭淵（http://aeb.iku4.com/）
　制作ツール：RPGツクールMV


◇概要
1〜2時間くらいで遊べるリソース管理型脱出RPGです。
14日間のうちにかぎられたリソースでキャラクターを強化し、
出口を塞ぐ「魔獣」を斃して「幽獄」から脱出することがゲームの目的になります。
本作は基本的にオートセーブですが、任意のタイミングでセーブすることもできます。
また、セーブファイルは一つのみです。


◇操作方法
　　　　　　【ゲームパッド】,【キーボード】,【マウス】
決定　　　：【A】,【Z,Enter,スペースキー】,【対象をクリック】
キャンセル：【B】,【X,Esc,テンキー0】,【右クリック】
メニュー　：【Y】,【X,Esc,テンキー0】,【右クリック】
前ページ　：【LB】,【Q,Page up】
次ページ　：【RB】,【W,Page down】

また、決定キーを押しっぱなしにすることで早送りができます。
戦闘中はコマンド入力中に「左カーソル」か「キャンセル」で
「逃げる」コマンドが出るようになります。

特殊操作
F2：FPSと描画モードの表示切り替え
F3：伸縮モード切り替え
F4：全画面モード切り替え
F5：リロード


◇FAQ
Q.戦闘中敵のターゲット指定ができなくなる
A.キーボードかゲームパッドで操作している場合、マウスカーソルにターゲットを取られている可能性があります。
マウスカーソルをゲーム画面外に移動させてプレイしてください。

Q.長時間プレイしていると重くなる
A.おそらくツクールMVの仕様です。
現在これといった対策がなく、お手数ですが再起動をお願いします。



◇難易度HARDについて
ver1.10より実装されました。
一度クリアすることで選択可能になります。
セーブデータの移行には「save」フォルダを移動させてください。
HARDモードには以下の特徴があります。

・新キャラ追加
・敵の強化
・聖霊出現確率低下
・ベッドの回復量制限(HPは最大値の半分のみ回復)

HARDモードはNORMALモードでは満足できないプレイヤー向けに、
このゲームの仕様を理解している前提の難易度設定になっています。
場合によっては「詰む」可能性も十分あります。
あらかじめご了承ください。



◇難易度EXTRAについて
ver1.13より実装されました。
HARDをクリアすることで選択可能になります。
セーブデータの移行には「save」フォルダを移動させてください。
EXTRAモードには以下の特徴があります。

・基本的にHARD準拠
・初期聖霊ランダム
・聖霊出現数制限(制限に達している場合触媒でも召喚不可)

EXTRAモードは縛りプレイのための難易度です。
特殊なプレイが楽しみたい方はご利用ください。
