『幽獄の14日間』ver0.81
　最終更新：2016/3/17 制作：饗庭淵（http://aeb.iku4.com/）
　制作ツール：RPGツクールMV


◇概要
1〜2時間くらいで終わるリソース管理型脱出RPGです。
14日間のうち、かぎられたリソースでキャラクターを強化し、
出口を塞ぐ「魔獣」を斃し、幽獄から脱出することがゲームの目的になります。
本作は基本的にオートセーブですが、任意のタイミングでセーブすることもできます。
セーブファイルは一つのみです。


◇操作方法
　　　　　　【ゲームパッド】,【キーボード】,【マウス】
決定　　　：【A】,【Z,Enter,スペースキー】,【対象をクリック】
キャンセル：【B】,【X,Esc,テンキー0】,【右クリック】
メニュー　：【Y】,【X,Esc,テンキー0】,【右クリック】
前ページ　：【LB】,【Q,Page up】
次ページ　：【RB】,【W,Page down】

また、決定キーを押しっぱなしにすることで早送りができます。

特殊操作
F2：FPSと描画モードの表示切り替え
F3：伸縮モード切り替え
F4：全画面モード切り替え
F5：リロード
